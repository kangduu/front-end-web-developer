将数组中 所有元素 排序，返回 排序后的数组。
        
        var arr = [1,2,5,7,5];
        var result = arr.sort(); // 不带参数 按默认方式排序
        console.log(result); // [ 1, 2, 5, 5, 7 ]
        
        console.log(arr); // [ 1, 2, 5, 5, 7 ] 改变原数组

        var arr = [1,2,3,undefined,8,9];
        console.log(arr.sort());
        
        var arr = [133,244,5555,7,5];
        var result = arr.sort();
        console.log(result); // [ 133, 244, 5, 5555, 7 ]
    
        自定义排序方式 -- arr.sort(function (a, b){ return >0 || <0 || =0 });
                function 决定了 a 和 b 在排好序的 数组中的先后顺序    
        a在前： 返回一个小于 0 的数 ---- 从小到大
        b在前： 返回一个大于 0 的数 ---- 从大到小
        相等： 返回一个等于 0 的数
