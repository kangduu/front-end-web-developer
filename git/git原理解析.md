### 本地库、缓冲区、中心库和工作区

- 本地库（Repository）：中心库的完整镜像（.git 目录）；
- 缓冲区（Index/Cache）：用于存放和整理待提交改动的空间，支持选择性提交（.git/index）；
- 中心库（Remote）：远程服务器（github、gitlab）管理下的git库；
- 工作区（Workspace）：本地开发环境下的文件内容。

### Git特点

- 支持离线工作，多点同步；
- 速度快；
- 使用40位哈希值标示版本号；
- 本地库、缓冲区、中心库三层结构；
- 轻量级的分支和标签；
- 主要基于命令，难上手；
- 占用空间大；
