一、数组直接量

        var arr = []
        var arr = [1,5,...]
        
        特别的
        1.[]中 可以是任何值
        2.省略的元素将被赋予 undefined 值
            var arr = [1,3,4,,5];
            console.log(arr[3]) // undefined
        3.最后一个元素不用添加 逗号
            var arr = [1,2,] // 只有两个元素

二、调用构造函数 Array()

        1. var arr = new Array() 等同于数组直接量
        
        2. var arr = new Array(10) // 参数为数组长度
        
            注意：创建一个数组可以用来预分配一个数组空间，但是 数组中没有存储值，这时甚至数组的索引属性“0”，“1”等还未定义。
        
        3. var arr = new Array(10，3，4，4，2) // 参数为数组值
