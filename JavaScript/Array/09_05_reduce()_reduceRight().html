语法：
        Array.reduce( function (initialV,v,i,a) {}, initial, this)
        Array.reduceRight( function (initialV,v,i,a) {}, initial, this)
        
        初始值和this对象都是可选的
        
    
        1、不指定初始值的情况下，将默认第一个元素为初始值
        
        2、数组只有一个值并且没有指定初始值 ， 简单的返回那个值(数组元素值)
            var a = [1];
            var re = a.reduce((init,v)=>{});
            console.log(re);// 1
        
        3、空数组且不带初始值，导致类型错误异常
            var a = [];
            var re = a.reduce((init,v)=>{
                console.log(init+v); //TypeError: Reduce of empty array with no initial value
            });
        
        4、空数组并且指定了初始值，简单的返回那个值(初始值)
            var a = [];
            var re = a.reduce((init,v)=>{},6);
            console.log(re);// 6
    
        reduceRight()的工作原理一样，唯一的不同就是：按照数组索引从高到低（从右到左）处理数组 ，
        
        *特别的： var a = [2,3,4] 计算 2^(3^4) ,就需要从右到左
