定义

            立即执行函数：立即调用的函数表达式（Immediately-Invoked Function Expression），简称 IIFE
            此类函数没有声明，在一次执行过后即释放。

写法

            (function (){}()); 官方（W3C）推荐
            (function (){})();
            
            注意:
                1、这只是常规写法，并不是只有这样写才称作立即执行函数
                2、上面两种写法最后的分号都是必须的。如果省略分号，遇到连着两个 IIFE ，可能就会报错
            
            特殊的有：任何让解释器以表达式来处理函数定义的方法，都能产生同样的效果
                var test = function (){}();
                //( + - 表示正负 )
                + function (){}();
                - function (){}();
                ! function (){}();
                ~ function (){}();
                true && function(){}();
                //逗号表达式
                0, function(){}();

作用

            1、适合做初始化工作，（即针对初始化功能的函数）
            2、目的一是不必为函数命名，避免了污染全局变量
            3、目的二是 IIFE 内部形成了一个单独的作用域，可以封装一些外部无法读取的私有变量。

特点

            1、没有函数名（也没必要写）
            2、唯一一个JavaScript提供的可以立即销毁的函数
            3、执行完过后再也找不到这个函数，或者说不再指向
            4、函数的引用不被保存
            5、通常情况下，只对匿名函数使用这种“立即执行的函数表达式”

区别

           在形式上，除了执行完就被释放以外没有任何区别

特别的

            只有表达式才能被执行符号执行
            能被函数执行符号执行的函数表达式，这个函数的名字就会被自动忽略，即放弃将函数的应用赋给变量
