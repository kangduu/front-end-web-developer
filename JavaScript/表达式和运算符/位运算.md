# 位运算符

> 你应该知道 `0.1+0.2 !== 0.3`

#### 有哪些位运算符?

`按位与 &` `按位或 |`  `按位异或 ^`  `按位非 ~`  `左移 <<`  `右移 >>>`  `用 0 补足的右移 >>>`  

**注意**： 所有的运算符针对的是 **整数** ，（若不是都会先转换为**整数**）



# 位运算符的妙用

1. 使用左移运算符 `<<` 计算2的次方

   ```
   console.log(1 << 2) // 4 2的2次方
   console.log(1 << 10) // 1024 2的10次方

   let a = 2e9;
   console.log(2000000000 << 1) // -294967296
   ```

2. 使用 按位异或 `^` 切换变量 0 和 1

   ```
   //before
   let a = 1;
   if (a) a = 0
   else a = 1
   console.log(a)  // a ===  0
   // or 
   let b = 1;
   b ? b = 1 : b = 0;
   console.log(b) // b ===  1

   // now 
   let c = 1;
   c ^= 1;
   console.log(c)  // c === 0
   ```

3. 使用按位与 `&` 判断奇偶性

   `偶数 & 1 = 0 `

   `奇数 & 1 = 1` 

   ```
   console.log(7 & 1) // 1 
   console.log(8 & 1) // 0
   ```

4. 使用 `!!` 将数字转为布尔值

   **所有的非0数组都是true，包括负数、浮点数。**

   ```
   console.log(!!7)   // true
   console.log(!!-2)   // true
   console.log(!!1.223)   // true
   console.log(!!0) //false
   ```

5. 使用 `~` ` >>` `<<` `>>>`得到整数

6. 使用 `^` 完成值的交换

7. 使用 `^` 判断符号是否相同

8. 使用 `^` 检查数字是否不相等
