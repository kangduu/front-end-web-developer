### 模块化的好处？

1. 有效避免命名冲突，解决污染全局问题
2. 更好的代码分离，模块依赖关系，按需加载
3. 更高的复用性
4. 高可维护性

### CommonJS和ES6 Module的区别？

1. CommonJS模块输出的是一个值的拷贝，而ES6模块输出的是值的引用；
2. CommonJS模块是在**运行时加载**，而ES6模块是在编译（es6编译为es5|3）时加载；
