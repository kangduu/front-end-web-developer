判定一个未知的对象 是否为数组：
        
        1、Array.isArray(o)
            Array.isArray([]) // true
            Array.isArray({}) // false
        
        2、typeof()
            typeof([]) // object
        
            注意：typeof 对于除了函数（返回function）以外的所有对象都返回 object。
        
        
        3、a instanceof b // a 是不是 b 的一个实例
